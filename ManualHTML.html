Manual de HTML

Construye páginas web desde cero con esta guía detallada.

Fundamentos

HTML es un lenguaje de marcado que estructura el contenido de las páginas web. Es fácil de aprender, pero extremadamente poderoso.

Etiquetas Clave

  • <html>: Define el inicio del documento.
  • <head>: Contiene metadatos y enlaces a recursos externos.
  • <body>: Incluye todo el contenido visible.

Ejemplo Práctico

<html>
  <head>
    <title>Mi Página Web</title>
  </head>
  <body>
    <h1>¡Hola Mundo!</h1>
    <p>Bienvenido a mi página.</p>
  </body>
</html>
